# Key Endpoints

- `GET /health` - Health check
- `POST /users/sign_up`, `POST /users/sign_in` - Auth
- `GET /users/me` - Current user profile
- `GET /api/profile`, `PATCH /api/profile` - Profile API
- `GET /admin/users` - Admin user list
- `GET /api/markets` - Markets CRUD
- `GET /api/markets/:market_id/articles` - Articles CRUD
- `GET /api/orders` - Orders
- `GET /` - Main page
